docker images- list images
docker ps - list containers

docker build -t ssl_spoofing_model .   - build current Dockerfile as ssl_spoofing_model
docker run -it ssl_spoofing_model /bin/bash    - run ssl_spoofing_model in interactive mode with open bash console

after opening bash console we are in the container and we need to navigate inside the correct 
conda activate SSL_Spoofing      or      source activate SSL_Spoofing


if we need to use external files we need to mount them when running the container:
docker run -it -v /path/to/local/Score_DF.txt:/app/Score_DF.txt -v /path/to/local/keys:/app/keys ssl_spoofing_model python evaluate_2021_DF.py /app/Score_DF.txt /app/keys eval
